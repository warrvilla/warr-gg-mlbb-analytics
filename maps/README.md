# Battle map images

Drop a **webp** for each map, lowercased with underscores:

| Map             | File                       |
|-----------------|----------------------------|
| Dangerous Grass | `dangerous_grass.webp`     |
| Broken Walls    | `broken_walls.webp`        |
| Flying Cloud    | `flying_cloud.webp`        |
| Expanding River | `expanding_river.webp`     |

Used as the AI Battle map-card backgrounds. Until a file exists the card
shows a dark gradient fallback (still selectable). Recommended ~400×300 webp.
